# 使用说明

## 从文件读取测试代码

程序现在支持从文件读取测试代码，而不是在代码中硬编码。

### 使用方法

#### 方法 1: 在 IDE 中按 F5 运行（推荐）

1. **直接按 F5**
   - 程序会自动编译并运行
   - 默认读取 `test.txt` 文件

2. **选择不同的配置**
   - 点击运行按钮旁边的下拉菜单
   - 可以选择：
     - **运行编译器（读取 test.txt）** - 读取 test.txt
     - **运行编译器（读取 test2.txt）** - 读取 test2.txt
     - **运行编译器（默认文件）** - 读取默认文件（test.txt）
     - **运行编译器（终端方式）** - 在外部终端运行

#### 方法 2: 命令行运行

**使用默认文件（test.txt）：**
```bash
.\compiler.exe
```

**指定文件：**
```bash
.\compiler.exe test.txt
.\compiler.exe test2.txt
.\compiler.exe mycode.txt
```

### 测试文件格式

测试文件应包含符合语法的代码。例如：

**test.txt:**
```
while ( true ) { 
    float b_flag = 1.5 ; 
    if_val = a_var ; 
    while ( b < 1 ) { 
        break ; 
    } 
    continue ; 
    b = a_var ++ ; 
}
```

### 错误处理

- 如果文件不存在，程序会显示错误信息并退出
- 如果文件为空，程序会显示错误信息并退出
- 如果代码有语法错误，程序会显示详细的错误信息

### 示例

1. **运行默认测试文件：**
   ```bash
   .\compiler.exe
   ```

2. **运行自定义测试文件：**
   ```bash
   .\compiler.exe test2.txt
   ```

3. **创建自己的测试文件：**
   - 创建一个 `.txt` 文件
   - 写入符合语法的代码
   - 运行：`.\compiler.exe 你的文件名.txt`

### 注意事项

- 文件路径可以是相对路径或绝对路径
- 文件内容会完整读取，包括换行和空格
- 代码必须符合编译器的语法规则
